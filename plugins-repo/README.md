## 📥 Установка

Плагины устанавливаются прямо из DARIA:

1. Открой **🛒 Магазин** на рабочем столе
2. Выбери плагин
3. Нажми **Установить**

---

## 🔧 Создание плагинов

Хочешь создать свой плагин? Читай [документацию](docs/PLUGINS.md)!

### Структура плагина

```
my-plugin/
├── plugin.yaml      # Манифест
├── main.py          # Код плагина
├── templates/
│   └── window.html  # UI окна
└── static/          # Статические файлы
```

### Отправка в каталог

1. Создай репозиторий с плагином
2. Открой Issue или PR в этом репозитории
3. Мы проверим и добавим в каталог!
