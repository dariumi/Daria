# Release Notes 0.9.0 — Morning Bloom

Дата релиза: 15 февраля 2026

## Что нового
- Даша стала заметно «живее» в desktop-среде: открывает нужные окна, меняет их размер и позицию, закрывает неактуальные.
- Появился новый монитор активности Даши с метриками процесса.
- Улучшен календарь (плитки + дни недели + маркеры).
- Улучшен старт-меню: компактные группы и аватар пользователя.
- Исправлена работа встроенного браузера с русскоязычными запросами.

## Качество общения
- Снижен спам-порог привлечения внимания.
- Улучшена мягкость ответа на сообщения поддержки.
- Исправлены артефакты форматирования в некоторых telegram-ответах.
- Усилена стабильность имени пользователя в памяти.

## Для разработчиков
- Добавлен API: `GET /api/daria/metrics`.
- Введён дополнительный HTTP trace (`before_request` / `after_request`) для отладки задержек.

## Совместимость
- Релиз обратно совместим с `0.8.x` конфигами.
- Рекомендуется обновить зависимости через `install.py` (особенно для tray/monitor/senses).
