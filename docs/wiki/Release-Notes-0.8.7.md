# Release Notes 0.8.7

Дата: 2026-02-14

## Что изменилось

- Реалистичный планировщик дел Даши: длительности, временные слоты, статус выполнения.
- Автономные занятия стали визуально наблюдаемыми: открываются реальные окна приложений.
- Личный дневник стал содержательнее: больше мыслей и рефлексии, меньше повторов.
- «Игры Даши» переименованы в «Игровой центр», улучшен Морской бой и исправлен цикл в Лабиринте.
- Снижена навязчивость attention/proactive, добавлен режим паузы по сообщению пользователя.
- Усилен контроль языка: меньше смешивания русского и английского.
- Добавлен календарь (API + окно интерфейса) и интеграция с планами.
- Telegram получил поддержку сценария участия в голосованиях через диалоговую команду.

## Для разработчиков

- Версия: `0.8.7`.
- Обновлён browser user-agent: `DARIA-Browser/0.8.7`.
