# Release Notes 0.8.5.1

Дата: 2026-02-13
Тип: Bugfix release
Кодовое имя: Quiet Pulse

## Что исправлено

- Стабилизированы SSE потоки уведомлений/логов, снижены ошибки разрыва chunked stream.
- Telegram plugin window больше не выбрасывает uncaught `Failed to fetch` при временно недоступном сервере.
- Улучшено UX при сетевых сбоях: понятные статусы вместо «тихих» падений.

## Что улучшено

- Wiki теперь отображает Markdown с читаемым стилем.
- Приложение «Самосознание» обновлено: добавлен редактор базовой self-инструкции.
- Self-инструкция сохраняется и влияет на системный промпт Даши.

## Новые API

- `GET /api/self/instruction`
- `POST /api/self/instruction`
