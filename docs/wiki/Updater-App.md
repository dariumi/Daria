# Updater App

Открой `⬆️ Обновления` на рабочем столе.

## Обновление ядра

1. Укажи `GitHub repo` и ветку.
2. Нажми `Проверить`.
3. Нажми `Обновить из GitHub`.
4. После успешного завершения перезапусти DARIA.

Либо используй `Обновить из архива` с локальным путем к `.zip/.tar.gz`.

## Обновление плагинов

- `Проверить плагины` показывает доступные апдейты.
- `Обновить всё` обновляет все плагины из каталога.
