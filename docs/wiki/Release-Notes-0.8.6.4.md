# Release Notes 0.8.6.4

Дата: 2026-02-14

## Что улучшили

- Морской бой получил полноценную визуализацию двух полей с цветовой индикацией клеток.
- Исправили поведение панели «Редактировать с Дашей» в файловом редакторе: теперь окно ведёт себя стабильно и без уезда контента.
- Укрепили `file-assist`, чтобы уменьшить ложные отказы при обычных пользовательских задачах.
- Подтвердили корректную навигацию в проводнике на уровень выше.
- В списке «Дела Даши» сделали компактную группировку повторяющихся выполненных дел (`×N`).

## Для разработчиков

- Версия проекта: `0.8.6.4`.
- Обновлены фронтовые и серверные версионные маркеры.
- Обновлён `User-Agent` встроенного браузера: `DARIA-Browser/0.8.6.4`.
