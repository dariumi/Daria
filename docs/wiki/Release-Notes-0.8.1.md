# Release Notes 0.8.1

## Исправлено

- Fixed: `GET /api/behavior` больше не падает 500.
- Fixed: `desktop.js` не вызывает `Cannot set properties of null` при открытии `Файлы`.
- Fixed: `voice-call` плагин больше не падает из-за отсутствующего `PluginAPI.get_user_profile`.
- Fixed: корректный `duration` при завершении звонка.
- Fixed: контекст диалога теперь включает последние пары `Пользователь/Даша`, не только сообщения пользователя.

## Улучшено

- Добавлен desktop-app `Обновления`:
  - обновление ядра из GitHub;
  - обновление ядра из локального архива;
  - переключатель автообновления.
- Добавлены API обновлений плагинов:
  - `GET /api/plugins/updates`
  - `POST /api/plugins/<plugin_id>/update`
  - `POST /api/plugins/update-all`
- Улучшен fallback-стиль ответов Даши: меньше статических шаблонов, больше учета контекста/настроения.
- `api/status` оптимизирован кэшем (быстрый отклик интерфейса).
- Плагин `Training`: добавлено обучение навыкам (например, 3D-скульптинг, рисунок).
